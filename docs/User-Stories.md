# User Stories for HumbleHire AI MVP

This document outlines the core functional requirements for the Minimum Viable Product (MVP) of HumbleHire AI, organized by Epics.

---

## 🔐 Epic 1 – Authentication

### Story 1: New User Registration
*   **As a** new user,
*   **I want to** register an account,
*   **So that** I can save my resumes.

### Story 2: Returning User Login
*   **As a** returning user,
*   **I want to** log in,
*   **So that** I can continue editing my resumes.

---

## 📝 Epic 2 – Resume Builder

### Story 3: Create Resume
*   **As a** job seeker,
*   **I want to** create a resume,
*   **So that** I can apply for jobs.

### Story 4: Edit Resume
*   **As a** job seeker,
*   **I want to** edit my resume,
*   **So that** I can keep it up to date.

### Story 5: Select Templates
*   **As a** job seeker,
*   **I want to** choose from professional templates,
*   **So that** my resume looks attractive.

### Story 6: Export to PDF
*   **As a** job seeker,
*   **I want to** download my resume as a PDF,
*   **So that** I can submit it to employers.

---

## 📊 Epic 3 – Dashboard

### Story 7: View Resumes List
*   **As a** user,
*   **I want to** see all my resumes,
*   **So that** I can manage them easily.

### Story 8: Delete Resume
*   **As a** user,
*   **I want to** delete an old resume,
*   **So that** I keep my dashboard organized.
