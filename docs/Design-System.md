# HumbleHire AI Design System

This document establishes the visual language, design tokens, and user experience patterns for HumbleHire AI. Maintaining this design system ensures UI consistency and scalable CSS architecture.

---

## 🎭 Brand Personality
HumbleHire AI balances cutting-edge technology with career reliability. The primary brand pillars are:
*   **Professional & Trustworthy:** Safe, serious, and secure for career growth.
*   **Modern & Minimal:** Clean interfaces, ample whitespace, and focused layouts.
*   **AI-Powered & Accessible:** Innovative utility without high-contrast futuristic visual clutter.

---

## 🎨 Color Palette

### Core Palette
*   **Primary (`#2563EB`):** Vibrant Blue. Communicates trust, reliability, and corporate professionalism.
*   **Secondary (`#1E40AF`):** Deep Blue. Used for hover states, active links, and prominent layout accents.
*   **Accent (`#14B8A6`):** Modern Teal. Infuses a subtle "AI/technological" spark into the product interface.

### Neutral Colors
*   **Background (`#FFFFFF`):** Base canvas layer for high clarity.
*   **Surface (`#F8FAFC`):** Soft Gray/Blue. Used for cards, alternating sections, and sidebar backgrounds.
*   **Border (`#E5E7EB`):** Clean architectural lines to divide interface regions seamlessly.
*   **Primary Text (`#111827`):** High-contrast off-black for clear, strain-free readability.
*   **Secondary Text (`#6B7280`):** Soft slate gray for descriptions, metadata, and placeholder text.

---

## 🔤 Typography
*   **Headings Font:** `Poppins`, sans-serif (Bold, geometric, clean aesthetic).
*   **Body Font:** `Inter`, sans-serif (Highly optimized for screen readability and UI microcopy).

---

## 📏 Spacing & Layout
*   **Section Padding:** `64px` to `96px` top/bottom separation to give landing sections room to breathe.
*   **Component Padding:** `16px` to `24px` internal spacing within elements.

---

## 📐 Border Radius
*   **Buttons:** `12px` (Soft, modern, interactive edges).
*   **Inputs:** `12px` (Matches button symmetry).
*   **Cards:** `16px` (Highly pronounced rounded containers for dashboard/preview assets).

---

## 👥 Shadows
*   **Standard Elevation:** `0 8px 30px rgba(0, 0, 0, .08)` (A subtle, smooth drop shadow providing soft surface depth without dramatic contrast).

---

## 🎛️ Interactive Components

### Buttons
*   **Primary:** Solid Primary Blue (`#2563EB`) text with white text, transitioning to Secondary Blue (`#1E40AF`) on hover.
*   **Secondary:** White surface background with a crisp light gray border, darkening text on interaction.

### Cards
*   **Base Style:** Clean White surface, `#E5E7EB` border execution, and `16px` rounded corners.
*   **Interaction:** Shifts slightly upward with a strengthened smooth shadow depth when hovered over.

### Forms
*   **Base Inputs:** White backgrounds accented by clear `#E5E7EB` borders. Focused fields display a soft Primary Blue glow ring.

---

## 🔮 Icons
*   **Library:** Lucide Icons / Heroicons.
*   **Execution:** Line-art minimalist stroke icons utilizing uniform vector sizing across all control menus.

---

## 🏗️ Landing Page Structural Architecture
The page layout acts as a strategic conversion funnel answering sequential consumer questions:

1.  **Navbar:** Clear company branding, essential navigation links, and a primary CTA tracking button.
2.  **Hero Section:** Strong value hook answering *"What is this platform?"*
3.  **Trusted Companies:** Social proof elements answering *"Can I rely on this tool?"*
4.  **Features Grid:** Deep feature overview answering *"Why should I use this over alternatives?"*
5.  **How It Works:** Step-by-step pathway answering *"How easy is it to set up and manage?"*
6.  **Resume Templates Preview:** Visual output mockups answering *"What high-quality results will I receive?"*
7.  **Testimonials:** User reviews answering *"Do other real professionals like this platform?"*
8.  **Pricing Preview:** Transparent cost columns answering *"How much does this option cost?"*
9.  **Call To Action (CTA):** Final high-conversion banner answering *"What concrete action should I take next?"*
10. **Footer:** Copyright markings, visual sitemaps, system settings, and policy links.
