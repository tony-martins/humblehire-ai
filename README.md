# HumbleHire AI

An AI-powered career platform that helps students, graduates, and professionals create ATS-friendly resumes, generate AI-assisted cover letters, and prepare for job opportunities.

---

## Project Vision

HumbleHire AI aims to simplify the job application process by providing modern tools that help users build professional resumes and improve their chances of getting hired.

---

## Problem Statement

Many job seekers struggle to:

- Create professional resumes
- Write compelling cover letters
- Understand what employers expect
- Build ATS-friendly resumes

HumbleHire AI solves these challenges through an easy-to-use web platform.

---
